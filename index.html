Ben Yehuda Word Frequency Checker


Created by Ben Gurion University students.

ABOUT THE PROJECT

This project was inspired by Google ngrams and meant to serve as a tool for checking word popularity over a timeline.

First, all texts were scanned using parts of speach tagger developed by Meni Adler. Link

Individual words were then counted and inserted into a database together with the year they were written,author and text. After the database was ready, it allowed interesting visualisation of word populerity, words by author, words by text.

Technologies used

  • Python
  • MySQL
  • PHP
  • Javascript
  • HTML
  • CSS
